⚡ MES APPLIS v0.9

Une collection d'outils et de jeux 100 % autonomes — chaque appli web tient dans un seul fichier HTML, sans installation (plus une appli musicale Windows), conçue et itérée maison, hébergée sur GitHub Pages. Cliquez une vignette (ou ▶) pour ouvrir l'appli en plein écran.

⚠ Page ouverte en file:// — ce fichier doit être dans le même dossier que les applis. Si les vignettes restent vides : python -m http.server puis http://localhost:8000.
