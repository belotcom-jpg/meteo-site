🔓 Contournements iframes

1. Mode Proxy (bouton dans la barre)

Passe les sites bloqués par corsproxy.io qui retire les headers X-Frame-Options. Fonctionne sur les pages HTML statiques (Bing, Mojeek HTML…), pas sur les apps React (IA).

2. 🤗 HuggingFace Spaces

HF autorise l'embed ! Entre user/space dans le champ vert. Des centaines de démos IA disponibles : Stable Diffusion, Whisper, MusicGen, LLaMA, Mistral…

3. Extension navigateur (solution ultime)

Supprime les headers côté navigateur — tous les sites fonctionnent en iframe, y compris ChatGPT, Claude, Gemini :

↓ Firefox — Ignore X-Frame-Headers

↓ Chrome/Edge — Ignore X-Frame-Headers
