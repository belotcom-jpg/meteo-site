ToDoXL v2.3 · API Microsoft Graph

Export et import complets de tes tâches Microsoft To Do (échéances, notes, sous-tâches) via l'API officielle. Tout reste dans ton navigateur.

🔐 Connexion Microsoft — via jeton Graph Explorer

  1. Ouvre developer.microsoft.com/graph/graph-explorer dans la fenêtre où to-do.live.com affiche tes tâches (même session = même compte), et connecte-toi (icône profil en haut à droite).
  2. Dans la barre de requête, mets https://graph.microsoft.com/v1.0/me/todo/lists et Exécuter la requête. Un 403 est normal la première fois.
  3. Onglet « Modifier les autorisations » → Tasks.ReadWrite → Consentir. Ré-exécute : la réponse doit lister tes vraies listes (pas juste « Flagged Emails » — sinon, mauvais compte).
  4. Onglet « Jeton d'accès » → icône copier 📋 → colle ci-dessous immédiatement (validité ~1 h).
⚙️ Mode avancé : app Azure personnelle (nécessite un tenant Azure — indisponible pour les comptes perso sans compte Azure)
  1. Va sur portal.azure.com → cherche « Inscriptions d'applications » (App registrations) → Nouvelle inscription.
  2. Nom : ToDoXL. Types de comptes pris en charge : choisis la dernière option, « … et comptes Microsoft personnels (Skype, Xbox) ».
  3. URI de redirection : plateforme « Application à page unique (SPA) », et colle l'adresse de cette page :
  4. Clique Inscrire, puis copie l'ID d'application (client) affiché en haut de la page et colle-le ci-dessous. Aucun secret nécessaire.
  5. À la première connexion, Microsoft demandera l'autorisation de lire/écrire tes tâches : accepte.
⚠️ La page doit être servie en https (GitHub Pages). Si tu obtiens « Selected user account does not exist in tenant 'Microsoft Services' », ton compte n'a pas accès au portail Azure : utilise le mode jeton ci-dessus.
