Plan de classe — aide

Page autonome pour gérer le plan en direct : interroger les élèves, donner des croix et piloter quelques outils de séance. Fonctionne hors-ligne, données sauvegardées dans le navigateur.

Le plan

  • Onglets = une classe chacun (5 par défaut). + ajoute, ✎ renomme, ✕ supprime.
  • Chaque case = un élève. Le plan se redimensionne pour tenir à l'écran sans défilement. Tableau/Bureau marquent le devant.

Les croix (clic sur un élève)

  • Clic gauche = croix négative : case blanc→noir (jusqu'à 10) + prénom prononcé.
  • Clic droit = croix positive : case jaune→bleu→vert (jusqu'à 10), sans son.
  • Compteurs indépendants ; badges +N (vert) / ×N (rouge) par élève ; jauges en bas = totaux de la classe. Réinitialiser = classe active ; Son coupe la voix.

Modifier le plan

  • Bouton Modifier le plan → les cases frétillent.
  • Glisser-déposer pour déplacer un élève (ses croix suivent), ✕ retire, case « + » ajoute.
  • Toucher une case = fiche élève : prénom + prononciation (ex. « Enael » → « é-na-el »), bouton Tester. Vide = lit le prénom tel quel.
  • + Rangée / + Colonne pour agrandir la grille.

Outils

  • Consigne (notes par classe), Minuteur (bip de fin), Humeur (auto selon le solde positives − négatives), Au hasard (tire un élève et l'annonce).

Import / Export

  • Importer Excel : .xlsx/.csv (couloirs et places vides conservés ; un fichier multi-feuilles peut recharger toutes les classes).
  • Exporter classes : .xlsx ré-importable. Exporter croix : .xlsx/.csv (Classe, Élève, Positives, Négatives, Solde).

Sauvegarde

Plans, prénoms, prononciations, croix et notes restent dans le navigateur (local ou GitHub Pages). La bibliothèque Excel se charge depuis Internet au premier import/export ; dans l'aperçu intégré de Claude, le stockage peut être bloqué (bascule en mémoire, sans sauvegarde entre rechargements).
