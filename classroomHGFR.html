ClassroomHGF — aide V1.0

ClassroomHGF est un tableau d'affichage pour la classe, à projeter. Tu organises ton affichage en pages, et sur chaque page tu poses librement des widgets (outils) que tu déplaces et redimensionnes. Par-dessus, un calque d'annotation plein écran permet de dessiner et de faire de la géométrie. Tout est enregistré dans le navigateur et fonctionne hors connexion.

En 4 gestes

  • + Widget → pose un outil sur la page. On le déplace par sa barre de titre, on le redimensionne par le coin, et la ✕ le retire.
  • Gérer les pages → renommer / supprimer / ajouter des pages (onglets en haut).
  • ✏️ Annoter → stylo, surligneur, laser, et 📐 Formes pour la géométrie, par-dessus toute la page.
  • Sauver / Charger → toute ta configuration dans un fichier .json.

Pages & configuration
  • Onglets de pages en haut : clique pour changer de page, glisse un onglet pour réordonner les pages. Gérer les pages affiche ✎ (renommer) et ✕ (supprimer) sur chaque onglet ; le + crée une page.
  • + Widget ajoute l'outil choisi à la page active (avec recherche 🔍 et récents en haut du menu). ⧉ dans l'en-tête d'un widget le duplique.
  • 📺 Présentation (ou touche P) : masque toute l'interface pour vidéoprojeter ; Échap pour sortir. Touche N : ouvrir le menu widgets. 🖨 Imprimer : page en PDF/papier.
  • 🔒 Tout : verrouille/déverrouille d'un coup tous les widgets de la page (pratique avant de projeter).
  • Lien kiosque : ajoute #page=2 à l'URL pour ouvrir directement la page 2, et #page=2&kiosque pour l'ouvrir en mode présentation (interface masquée).
  • 🎨 Thème → 🖼 Image de fond : choisis une photo comme fond de la page (compressée automatiquement) ; « ✕ Retirer l’image » pour revenir au thème.
  • ⤢ Agrandir : agrandit la zone de travail par paliers (1×1 → 2×2 → 3×3). La page s'étend à droite et vers le bas (défilement), avec des repères qui marquent chaque « écran ». Chaque page garde son propre niveau.
  • ⛶ Plein écran : passe l'appli en plein écran pour vidéoprojeter (re-cliquer pour quitter).
  • 🔓 / 🔒 sur chaque widget : verrouille sa position (impossible de le déplacer ou redimensionner par erreur).
  • Ctrl+Z / Ctrl+Y : annuler / rétablir (ajout, déplacement, redimensionnement, suppression de widgets, agrandissement de page).
  • Thème de page : change l'ambiance de fond.
  • Sauver / Charger la config : exporte/importe tout (pages, widgets, contenus) en .json, pour sauvegarder ou passer d'un PC à l'autre. On peut aussi exporter la page active seule pour partager une mise en page à un collègue (l'import l'ajoute comme nouvelle page).
  • Sans rien faire, tout est auto-enregistré dans le navigateur. Vider les données du navigateur efface la configuration → garde un .json de secours.

🐢 Atelier code
  • Deux modes : Blocs (palette maison, fonctionne hors-ligne) et Blockly (vrais blocs à glisser, chargé depuis Internet à la 1re ouverture).
  • Une tortue se déplace sur la scène et laisse une trace. Blocs disponibles : avancer/reculer, tourner ↻/↺, crayon haut/bas, couleur, épaisseur, dire, effacer, Répéter n fois, Répéter jusqu'au bord et Si … (touche le bord). 🖼 PNG exporte le dessin.
  • Mode Blocs : clique un bloc pour l'ajouter ; règle les nombres directement ; ↑ ↓ ✕ pour ordonner/supprimer ; ＋ ajouter ici pour remplir une boucle/un Si. ▶ Exécuter, ↺ réinitialise la scène, 🗑 efface le programme, curseur Vitesse.
  • Mode Blockly : palettes Mouvement / Stylo / Contrôle (boucles, si, tant que) / Capteurs / Nombres ; le programme génère et anime la même tortue. Une sécurité stoppe les boucles infinies.
  • Le programme est sauvegardé avec la config (blocs maison et montage Blockly). Hors-ligne, seul le mode Blocs est disponible.

⚡ Questions flash
  • Génère un QF de 9 questions dont le type change à chaque tirage (piochées au hasard parmi vrai/faux, calculs, fractions, chiffre des unités, suites, opérateur manquant, minutes, romains, unités, plus petit/grand, moyenne, zéros inutiles…).
  • Niveaux 6e → 3e : ajustent la taille des nombres. 🔄 Nouvelles tire un nouveau QF (mémorisé, il ne change pas au rechargement). 👁 Réponses affiche/masque le corrigé.
  • Idéal en rituel de début d'heure ; passe le widget en plein écran pour le vidéoprojeter.
📖 Cours à révélation
  • Mode ✎ Éditer : rédige ton cours, sépare les étapes par une ligne ---, masque une réponse avec [[doubles crochets]].
  • Mode ▶ Présenter : dévoile les étapes une à une (◀ / Suivant ▶ / Tout) ; clique une zone masquée pour la révéler. Parfait pour reprendre tes cours Google Docs (copier-coller le texte).
🔳 QR code
  • Affiche un QR code depuis une URL ou un texte : les élèves le scannent (lien d'activité, site…). Le générateur se télécharge à la 1ʳᵉ ouverture (nécessite Internet).
▦ Tableur
  • Deux modes (onglets dans le widget) : Tableau (saisie simple de texte/nombres) et Formules (tableur avec calculs).
  • + ligne / + col / − ligne / − col pour redimensionner la grille. Clique une cellule pour saisir ; Entrée descend, Tab va à droite, Échap annule la saisie.
  • En mode Formules, une cellule qui commence par = est calculée : références (A1, B2), plages (A1:A5), opérations + − * / ^ et parenthèses.
  • Fonctions : SOMME, MOYENNE, MIN, MAX, NB, NBVAL, PRODUIT, SI, ABS, RACINE, PUISSANCE, ARRONDI, ENT, MOD, PI (noms anglais aussi acceptés).
  • Convention française : séparateur d'arguments ; et décimale , — ex. =SI(A1>=10;"ok";"-"), =MOYENNE(B1:B9). Le contenu est sauvegardé avec la config.
  • ⬆ Importer un fichier .xlsx ou .csv (ex. ton classroom Google Sheet exporté). ⬇ Excel / ⬇ CSV pour exporter. Recopie ↓ copie la cellule sélectionnée vers le bas en ajustant les références (B1=A1*2 → B2=A2*2…).
✏️ Annotation & géométrie (par-dessus la page)

Le bouton ✏️ Annoter ouvre un calque plein écran. La palette propose le dessin libre (stylo, surligneur, laser, gomme, couleur, épaisseur) et 📐 Formes pour la géométrie.

Tracés (1 carreau = 1 unité) :

  • Objets : point (étiqueté A, B, C…), segment, demi-droite, droite, vecteur, cercle, compas, arc et secteur, ellipse, rectangle, carré, triangles (quelconque, équilatéral, isocèle, rectangle), polygone régulier (3–12) et libre, angle droit ⦜ et angle mesuré ∠°.
  • Constructions : milieu, médiatrice, perpendiculaire, parallèle, texte mathématique.
  • Transformations : translation, rotation, symétrie axiale, symétrie centrale (l'image est créée, l'original conservé).

Mises en forme & mesures :

  • Codages : marques d'égalité, angles droits, angles égaux — bouton Codage, ou 🔎 Scan codage pour tout coder.
  • Long. (longueurs), Coord. (coordonnées, repère au centre), 🎨 Colorier (couleur, opacité, motif).
  • Repère, Grille, Aimanter. Figures : 💾 Enregistrer / Charger (navigateur), ⬇ Exporter / ⬆ Importer en .json.
⚠️ Les figures sont statiques : déplacer un point ne recalcule pas la construction (ce n'est pas GeoGebra). Pour partager entre PC, passe par l'export .json.
⌨️ Raccourcis clavier
  • Échap : ferme l'aide / une fenêtre / l'annotation, ou annule un polygone en cours.
  • Entrée : termine un polygone libre. Suppr / ⌫ : supprime la forme sélectionnée.
  • Ctrl/Cmd + C / V : copier / coller une forme.
